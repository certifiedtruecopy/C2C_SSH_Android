* برای اندروید

* حذف کاربر

* محدودیت های ترافیکی
* اضافه شدن قابلیت تونل بین دو سرور

* نصب انجینیکس
* نصب قالب وب سایت

* نمایش تاریخ ایجاد و انقضای یک کاربر.
* بلاک کردن دامین دامنه ها
* با استفاده از دامین به سرور دسترسی پیدا کنید.
